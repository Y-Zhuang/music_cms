#### CMS音乐管理系统，主要是在RBAC权限系统的基础上增加对歌曲与歌单的CRUD操作，以及歌曲的上传和在线试听等功能。

****

#### 前端 music_cms_client

- Vue
- Vue-router
- Vuex
- ElementUI
- axios

#### 后端 music_cms_service

- SpringBoot
- MySQL
- Jwt
- 阿里云OSS对象存储
